Project Two: Unreal

Climb On!

One Week Prototype

About the Project

This project was created in one week for my Game Design 2 class in GameMaker Studio. We were given the prompt/mechanic "ladders" and were told to make a sidescroller. In my prototype, the player moves 4 independent "limbs" (two hands and two feet). The goal of the game is to reach the top "hold" with both hands. My interpretation of the prompt, rock climbing, uses the given mechanic to allow the player to stop their movement at different verticalities. This is helpful to the player as the lose condition is letting your "hips" (the box in the center of the screen) touch the wall. The art theme for this project was "monochrome", which is why there is only black and three shades of blue.

What I Would Change

The one level in this project is very difficult, which is by far the largest piece of feedback I recieved. Additionally, I would also add more feedback for the player (highlight which limb they are currently using, add audio feedback, and create an actual UI).

For this class, we have an opportunity to take one of our prototypes and expand upon it via adding more levels, changing the theme (to a different in-class theme), and spending more time on it. I will be taking the feedback mentioned above and implementing it into my game. Furthermore, I have refactored the code to run better in GameMaker Studio. I have changed the theme from monochrome to friendly robots, meaning that inspiration will be more obvious and an actual story.
